# Terwe2a w 3asha - Restaurant Menu Website

A modern, elegant restaurant menu website for Terwe2a w 3asha bakery & restaurant.

## Quick Start

### Option 1: Live Server Extension (Recommended - No Terminal Needed)

1. Install the **Live Server** extension in VS Code (by Ritwick Dey)
2. Right-click on `index.html`
3. Select **"Open with Live Server"**
4. Your browser will open automatically with the site running!

### Option 2: Node.js Server

1. Make sure Node.js is installed on your computer
2. Press **F5** in VS Code (or go to Run > Start Debugging)
3. The server will start and Chrome will open automatically

### Option 3: Manual Server Start

If you prefer to start the server manually:
- Open terminal in VS Code (Ctrl + `)
- Run: `node server.js`
- Open browser to: `http://localhost:8080`
